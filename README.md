# Dash-Tempo
A haunted mansion themed rhythm game created using Java, using hashmap and creating different classes of properties for different physics properties used throughout the game. Used Procreate to hand-draw the backgrounds and multiple sprites of the game.

3 levels are created in order to finish off the final boss. The player can pick up different abilities during the game in order to increase their chances of successfully jumping on the required blocks to avoid the dangerous bullets ahead.

Features:
Gameplay
Tutorial of Commands
Editable Settings
Backstory
